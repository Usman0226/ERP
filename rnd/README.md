# Research & Development (R&D)

Manages university research: projects, grants, researchers, publications, patents, datasets, and collaborations.

- API base: `/api/v1/rnd/`
- Auth: JWT (see `api/auth/token/`)

Key endpoints (via DRF router):
- `researchers/`
- `grants/`
- `projects/`
- `publications/`
- `patents/`
- `datasets/`
- `collaborations/`
